# AIM

To write prompts for the following Prompt Engineering techniques and compare the outputs generated using different AI tools with evaluation using a rubric method.

1. Comparative Analysis Prompt

# SCENARIO

The manufacturing industry is looking to reduce manual monitoring and increase efficiency through automation. The system uses IoT devices and embedded controllers to automate equipment, monitor performance, and enable predictive maintenance.

---

# TARGET AUDIENCE

Manufacturing companies in sectors such as:
- Automotive
- Electronics
- Food Processing

---

# MAIN OBJECTIVES

- Improve production efficiency by 30%
- Minimize machinery downtime using predictive maintenance
- Enable real-time monitoring and remote control
- Reduce energy consumption through optimization

---

# AI TOOLS USED

| AI Tool | Purpose |
|---|---|
| ChatGPT | Conversational AI response generation |
| Gemini | Comparative AI-generated responses |

---

# 1. COMPARATIVE ANALYSIS PROMPT

## Definition
A comparative analysis prompt compares two or more concepts by identifying similarities and differences.

---

## Example Prompt
```text
Compare and contrast IoT-based automation and traditional manufacturing systems.
```

---

## ChatGPT Output
```text
IoT-based automation uses smart sensors, real-time monitoring, and predictive maintenance to improve efficiency. Traditional manufacturing systems rely heavily on manual monitoring and periodic maintenance. IoT systems reduce downtime and improve energy efficiency, while traditional systems are slower and less adaptive.
```

---

## Gemini Output
```text
Traditional manufacturing systems depend on manual processes and fixed schedules, whereas IoT-based automation enables connected devices to exchange data in real time. IoT improves productivity, monitoring, and predictive maintenance compared to traditional methods.
```

---

# 2. EXPERIENTIAL PERSPECTIVE PROMPT

## Definition
This prompt asks the AI to respond from a particular person's perspective or experience.

---

## Example Prompt
```text
As a factory manager, explain the benefits of using IoT automation in manufacturing plants.
```

---

## ChatGPT Output
```text
As a factory manager, IoT automation helps monitor machines continuously, reduces maintenance costs, improves productivity, and minimizes downtime through predictive maintenance.
```

---

## Gemini Output
```text
From a factory manager’s perspective, IoT automation improves operational efficiency, provides real-time equipment monitoring, and helps optimize production schedules effectively.
```

---

# 3. EVERYDAY FUNCTIONING PROMPTS

## Definition
These prompts explain how technology is used in everyday applications.

---

## Example Prompt
```text
Describe how IoT technology is used in everyday life.
```

---

## ChatGPT Output
```text
IoT technology is used in smart homes, fitness trackers, smart TVs, and automated security systems. These devices collect and exchange data to improve convenience and efficiency.
```

---

## Gemini Output
```text
IoT is commonly used in smart watches, home automation systems, connected vehicles, and healthcare monitoring devices that communicate through the internet.
```

---

# 4. UNIVERSAL PROMPT STRUCTURES

## Definition
Universal prompt structures use common templates that can work across many topics.

---

## Example Prompt
```text
Explain predictive maintenance in simple terms with an example.
```

---

## ChatGPT Output
```text
Predictive maintenance is a method of checking machine conditions using sensors to predict failures before they happen. For example, a sensor in a factory motor can detect overheating and alert technicians before the machine breaks down.
```

---

## Gemini Output
```text
Predictive maintenance uses data and sensors to monitor equipment health and prevent sudden failures. For instance, vibration sensors can identify machine issues early.
```

---

# 5. PROMPT REFINEMENTS

## Definition
Prompt refinement improves prompts by adding clarity and details for better AI responses.

---

## Initial Prompt
```text
Explain automation.
```

---

## Refined Prompt
```text
Explain industrial automation in manufacturing with advantages and real-time examples.
```

---

## ChatGPT Output
```text
Industrial automation uses machines, sensors, and control systems to perform manufacturing tasks automatically. Advantages include increased productivity, reduced labor costs, and improved product quality. Examples include robotic assembly lines and automated packaging systems.
```

---

## Gemini Output
```text
Industrial automation refers to the use of control systems and robotics in factories. It improves efficiency, accuracy, and safety. Examples include CNC machines and automated conveyor systems.
```

---
